# コメ゙カン

## 1. 概要 (Concept)

**コメカン** は、増え続けるコスメを簡単に管理し、その組み合わせ（メイクレシピ）を共有することで、毎日のメイクをもっと楽しくするアプリを目指して作成されています。

* **Manage (管理):** バーコードで瞬時に登録。手持ちのアイテムを可視化し、買いすぎを防ぎます。
* **Social (共有):** テンプレートを使って、着せ替えゲーム感覚で気軽にメイクレシピを共有。他の人の使い方を参考にできます。

## 2. 解決する課題

* **課題1: 在庫の不透明性**
    * 気がついたら同じような色のリップばっかり...
    * → **[解決策]** 統計機能で手持ちのアイテム比率をグラフ化。自分の「癖」を把握し、賢く買い物できます。
* **課題2: 活用方法のマンネリ化**
    * 持ってるけど使ってないコスメ（＝死蔵コスメ）がある
    * → **[解決策]** メイクレシピの共有機能。他の人があなたのコスメ（と同じアイテム）をどう使っているか知ることで、新しい組み合わせを発見できます。

## 3. 主な機能

### Manage（管理）

* **バーコード登録:** JANコードをスキャンするだけで商品情報を自動入力。
* **手動登録:** データベース検索・サジェストによる手動登録・新規作成にも対応。
* **マイリスト（コスメ棚）:**
    * 「ベースメイク」「アイメイク」などの大カテゴリ・「ファンデーション」「アイシャドウ」などの小カテゴリで自動分類。
    * 「#ブルベ夏」「#お気に入り」など、自分だけのカスタムタグで自由に整理。
    * 購入日の管理も可能。
* **統計ダッシュボード:**
    * カテゴリ別、色別などで所持アイテムの割合をグラフ化。家計簿アプリのように、自分のコスメ資産を可視化します。

### Social（共有）

* **メイクレシピ共有:**
    * 「今日のメイク」を簡単にシェアできる独自SNS。
    * テンプレートにマイリストからコスメをポチポチはめるだけで、ビジュアル的に美しいレシピが完成します。
    * 実際の自撮り写真も添付可能。
* **SNS連携:**
    * こだわりの投稿はInstagramで。インスタの投稿をアプリ内に直接埋め込んで表示できます。
* **新作コスメフィード:**
    * 新発売のコスメ情報が流れてくるタイムラインで、次の「欲しい！」に出会えます。

## 4. 技術スタック（使用技術）

### 📱 フロントエンド (Mobile)
* **Flutter**

### ☁️ バックエンド (Server-Side)
* **言語:** Java 17+ (Spring Boot 3)
* **実行環境:** AWS Elastic Beanstalk
* **データベース:** Amazon RDS (MySQL or PostgreSQL)
* **ストレージ:** Amazon S3 (画像保存用)
* **認証:** Amazon Cognito
* **API:** REST API

### 🧩 外部API
* 商品検索API (バーコード情報取得のため)
* Instagram Basic Display API
